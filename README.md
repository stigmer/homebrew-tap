# Stigmer Homebrew Tap

Official Homebrew tap for [Stigmer](https://github.com/stigmer/stigmer) - AI-powered workflow automation with local LLMs.

## Installation

```bash
brew tap stigmer/tap
brew install stigmer
```

## Update

```bash
brew update
brew upgrade stigmer
```

## Uninstall

```bash
brew uninstall stigmer
brew untap stigmer/tap
```

## About

This tap is automatically updated by the [release workflow](https://github.com/stigmer/stigmer/blob/main/.github/workflows/release-embedded.yml) when new versions are released.
